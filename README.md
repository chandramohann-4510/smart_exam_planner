## 🚀 What Is This?

Smart Exam Planner is a **fully offline, single-file web app** built for ALIET 2026 students to track their exam schedule, monitor study progress, and access study materials — all from one beautifully designed page. No installation. No account. Just open the HTML file in any browser and go.

---

## ✨ Features

| Feature | Description |
|---|---|
| ⏱ Live Countdown Bar | Sticky top bar with real-time countdown to the next upcoming exam |
| 📋 Exam Cards | Cards for all 11 exams across 3 phases with dates, times & syllabus |
| ☑️ Syllabus Checklist | Tick off units/topics as you study — progress saves automatically |
| 📊 Progress Tracker | Overall % progress bar + per-subject mini progress bars |
| 📚 Study Resources | One-click button to open the Google Drive study materials folder |
| 🎇 Animated UI | Floating particles, glowing effects, sparkles — pure CSS/JS |
| 💾 Auto-Save | Study progress saved in browser localStorage — persists on refresh |
| 📱 Responsive | Works on mobile, tablet, and desktop |

## 🛠 How to Use

1. **Download** the `smart_exam_planner_v2.html` file
2. **Open** it in any modern browser (Chrome, Firefox, Edge, Safari)
3. **Check off** topics as you study them — progress saves automatically
4. **Click** the Study Resources button to open the Google Drive folder
5. **Share** the file with classmates — each person's progress is saved only on their own device

> ⚠️ Do NOT open via `file://` with strict browser settings that block localStorage. Just double-click the file normally.
